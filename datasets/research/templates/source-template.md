## Template Usage Guidelines

### Metadata Field Guidance

**Decision Relevance Levels:**
- **High**: Directly informs a current strategic decision
- **Medium**: Provides supporting context for strategic decisions  
- **Low**: Background information, may be useful for future decisions

**Data Confidence Levels:**
- **High**: Authoritative source, recent data, verified methodology
- **Medium**: Reliable source but some limitations (age, methodology, scope)
- **Low**: Useful insights but significant limitations or uncertainties

### Optional Sections (Include When Relevant)

**Economic Implications**: Include when source contains:
- Revenue or cost data
- Margin analysis
- Investment requirements
- Pricing benchmarks
- Financial performance metrics

**Customer Impact Analysis**: Include when source contains:
- User behavior patterns
- Market segment analysis
- Adoption factors
- Customer experience insights
- Usage patterns or preferences

### Section Focus Guidelines

- **Key Insights**: Focus on strategic takeaways, not comprehensive summaries
- **Strategic Applications**: Be explicit about which decisions this supports
- **Quotes**: Select for citation value, not comprehensive coverage
- **Economic/Customer sections**: Quantify impacts when possible
