## Purpose

Establish what to measure for a new product, feature, or initiative through systematic analysis from company mission to specific metrics.

## Required Inputs

- Company/product mission statement
- Business model type
- Feature/product description
- Target user segment

## Workflow Sequence

1. North Star Alignment → Identify company-level metrics
2. Funnel-Based Metric Mapping → Decompose user journey
3. Proxy Metric Selection → Create measurable indicators
4. Trade-off Evaluation → Identify counter-metrics
5. Prioritization → Select 1-2 primary metrics

## Output

- 3-7 candidate metrics with formulas
- 1-2 prioritized primary metrics
- Counter-metrics identified
- Funnel mapping documented
- Monitoring plan established

## Time Estimate

70-90 minutes total
